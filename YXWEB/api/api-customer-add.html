客户API

  • 所有客户
  • 新增客户

/customer/create

通过新增客户接口往"二当家"添加客户信息

URL

http://edj.yunxiaokeji.com/customer/create

支持格式

JSON

HTTP请求方式

POST

请求参数

参数 必选 类型及范围 说明
cid 是 string 公司身份
key 是 string 公司密钥
customer 是 json 客户json实体

客户json实体

{

    "name": "客户名称",
    "sourceid": "客户来源ID",
    "type": 客户类型 0：个人客户；1：企业客户,
    "industryid": "行业ID",
    "extent": 客户公司规模 1:0-49 2：50-99 3：100-199 4：200-499 5：500-999 6：1000+ ,
    "contactname": "联系人",
    "mobilephone": "联系电话",
    "email": "邮箱",
    "job": "职位",
    "description": "备注"

}

返回结果

{

    status:1

}
